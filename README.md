# Տորթերի խանութ — Bakery E-commerce Monorepo

Professional custom e-commerce platform for a cake shop (bakery), built as a monorepo with three applications.

## Structure

| Folder    | Stack                          | Purpose                    |
|-----------|--------------------------------|----------------------------|
| `backend` | Node.js, Express, TypeScript   | REST API + MongoDB         |
| `client`  | Next.js, Zustand, Tailwind     | Public website (Armenian)  |
| `admin`   | React, Vite, Ant Design        | Admin panel (Armenian)     |

## Requirements

- Node.js 20+
- MongoDB

## Development

Each application is developed and run independently from its own folder. Setup instructions will be added as each part is implemented.
